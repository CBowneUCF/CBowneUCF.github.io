My Useful Unity Scripts
A collection of often-reused Scripts I've written to make life easier in Unity.
Written entirely by me.

Rage Rooster
A comedic 3D Beat-em-up Metroidvania where a furious rooster plows through a goofy alien threat.
I scripted a large portion of the code base, including the GameObject-based State Machine system.

Cake Toppers
A cutesy sweet cake-baking game for IOS where the player must complete orders of increasing complexity.
I scripted the majority of the code base.

Best Served Raw
A 3D 3rd-Person Beat-em-up starring a Shinobi turned Sushi Chef, haunted by his past.
I scripted the majority of the code base, excluding the common enemies.

SCP - Line of Sight
A simple First-Person game where the player must face down a statue that moves when you’re not looking.
I scripted the majority of the code base.

Pac-Man - Haunted Knights
A UCF-themed Pac-Man clone featuring a Pegasus being hunted by 4 haunted suits of armor.
